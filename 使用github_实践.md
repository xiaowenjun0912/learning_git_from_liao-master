- **如何进行 pull request ?**
首先将你相中的项目，fork 到自己的仓库，然后运行 gitbash clone 到本地进行修改，提交并 push 到远程仓库，然后在自己的仓库发起 pull request 。OK，等待项目管理者处理。
进入自己 fork 的仓库，点击 New pull request 按钮，如下图
